
## Data Availability for:
**A contrastive graph-transformer network model for multimodal sentiment analysis**

---

## 1. Overview

This repository provides access to the datasets used in the study entitled:

**"A contrastive graph-transformer network model for multimodal sentiment analysis"**

The purpose of this repository is to ensure transparency, reproducibility, and compliance with the journal's data availability requirements.

---

## 2. Datasets

The following publicly available datasets were used in this study:

- **The CMU-Multimodal Opinion Sentiment and Emotion Intensity (CMU-MOSI)** *(Zadeh et al., 2016)*: This dataset, developed in English, includes audio, text, and video modalities compiled from 2,199 annotated video segments collected from YouTube monologue movie reviews. It offers a focused approach to studying sentiment detection within the context of film critiques.

- **The CMU-Multimodal Sentiment Analysis (CMU-MOSEI)** *(Bagher Zadeh et al., 2018)*: This dataset is an extension of CMU-MOSI, incorporating the same modalities of audio, text, and video from YouTube videos. It has a broader scope, covering a wider range of topics, and is more substantial in size, with 23,453 annotated video segments.

- **The Chinese Multimodal Sentiment Analysis Dataset (CH-SIMS)** *(Yu et al., 2020)*:  This dataset is a publicly available benchmark for Chinese multimodal sentiment analysis (MSA), widely used for training and evaluating MSA models. It comprises 60 original videos segmented into 2,281 high-quality clips, covering diverse speakers with varied backgrounds and age ranges. Each clip is annotated with sentiment polarity labels, including positive, negative, and neutral categories.

These datasets are widely used benchmarks in multimodal sentiment analysis research.
